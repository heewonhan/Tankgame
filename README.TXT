========================
  CSC413 Tank War Game
========================
This game is two players versus each other.
To run the game, download and execute Tankgame.jar in jar folder.
Player 1: left arrow key, up arrow key(forward), right arrow key, down arrow key(back), and enter(shoot).
Player 2: a key(left), w key(forward), d key(right), s key(back), and q key(shoot).
Developed by JDK 1.8 and Netbeans IDE 8.2
